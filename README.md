# callingmama

## 집에 가고 싶은데 가기 애매한 상황일 때 어플을 실행하면 미리 설정해둔 사람의 이름과 메세지 내용이 10초뒤에 스마트폰 알림창에 나타나게하여 그 자리를 빠져나갈 수 있는 서비스입니다.
## 백그라운드 알림을 구현하여 폰이 꺼져도 실행되는 무박2일 해커톤에서 진행한 프로젝트입니다.
